# YIM_Project
Repository for Published YIM Project Materials

## License and Terms of Use

This repository uses a multi-license model to ensure maximum reproducibility and open-science compliance:

* **Research Papers & Written Content:** All academic papers, markdown files, PDFs, and diagrams are licensed under the [Creative Commons Attribution 4.0 International (CC-BY 4.0)](https://creativecommons.org/licenses/by/4.0/) license. 
* **Data & Audit Trails:** All verification reports, SHA256 hashes, and data files (including those hosted on Zenodo) are dedicated to the public domain under [CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/). Note: Certain audit trails are hosted on Zenodo under "Restricted Access" and are available upon request.
* **Source Code:** Any scripts, automation tools, or code snippets included in this repository are licensed under the [MIT License](https://opensource.org/licenses/MIT).

When citing these works, please attribute the **YIM Project Research Initiative** and link to this repository or the respective Zenodo DOI.



## How to Cite
If you use the data, frameworks, or code in this repository, please cite the Zenodo archive to ensure persistent linking:
